# Guides to Q

Maana product documentation

## Build

Install mkDocs using [PIP](https://packaging.python.org/tutorials/installing-packages/):

```bash
pip install mkdocs
```

To produce a static site, within the root direcotry, simply:

```bash
mkdocs build
```

## Serve

To test the site:

```bash
mkdocs serve
```

## Deploy

To update GitHub's live view:

```bash
mkdocs gh-deploy
```
